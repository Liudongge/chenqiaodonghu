# 安装
**安装cnpm**</br>
在命令行中输入 npm install -g cnpm --registry=http://registry.npm.taobao.org

**安装项目所需的依赖**</br>
要安装依赖包，首先cd到项目文件夹然后运行命令 cnpm install

**启动服务**</br>
npm run dev
